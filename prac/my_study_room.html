HTML 정의

HTML(Hypertext Markup Language): 웹페이지와 그 내용을 구조화하기 위해 사용하는 코드

HTML 기본 구성 순서 및 개념

HTML 기본 구성 순서

  1. !DOCTYPE html
    • 문서의 형태가 어떤 것인지 알려줌
    • DOCTYPE(Document Type Declaration): 문서 형식 선언
    • HTML(Hypertext Markup Language): 웹페이지와 그 내용을 구조화하기 위해 사용하는 코드
  2. html lang="ko"
    • html 태그를 넣어 전체 틀을 잡고 하위 태그로 head와 body 기입
    • html 뒤에 lang(language: 언어)를 적어 언어를 설정해준다.
    1. head
      • html 본문이 아닌
        • meta charset="UFD-8"
    2. body
